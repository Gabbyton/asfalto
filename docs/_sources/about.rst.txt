*****
About
*****

``ASFALTO`` is an complementary package to the `CEMENTO <https://cwru-sdle.github.io/CEMENTO/index.html>`_ package, and is used to accelerate ontology development by streamlining template creation and enabling bottom-up abstraction.

``ASFALTO`` stands for the Abstraction Scaffolding Framework and Automated Linker of Templates for Ontologies.
